## 📦 Contribution

1. Clone the repository

2. At root level run

```bash
pnpm i
```

3. Change the at /apps/cli

4. Write test for your change or test them with

```bash
pnpm test
```

if you want to change snapshot run

```bash
pnpm test:snapshot:update
```
